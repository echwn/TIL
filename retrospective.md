## 1. Today I learned

### 1-1. git에서 관리하는 세 영역

- 작업 디렉토리 (Working directory)
  - 현재 편집 중인 파일이 저장되는 영역
- 스테이징 영역 (Staging area)
  - 저장소에 저장할 변경 사항을 보관하는 임시 저장소
- .git 디렉터리 (Repository)
  - 모든 작업 내역이 영구히 저장되는 저장소

### 1-2. JavaScript 표현식(expression)과 연산자(operator)

```// 논리 연산
true || false;
true && false;

// 진리값으로 취급하기
1 || 0;
1 && 0;
```
